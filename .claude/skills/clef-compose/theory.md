## ABC 记谱法参考

ABC 记谱法是 clef-compose v2 的核心输出格式，所有 Agent 使用 ABC 表示音符、和弦和节奏。

### ABC 头部字段

| 字段 | 格式 | 说明 | 示例 |
|------|------|------|------|
| X: | `X:<编号>` | 曲目编号（必须第一行） | `X:1` |
| T: | `T:<标题>` | 曲目标题 | `T:Battle Theme` |
| M: | `M:<拍号>` | 拍号 | `M:4/4` `M:3/4` `M:6/8` |
| L: | `L:<时值>` | 默认音符时值 | `L:1/4`（四分音符） `L:1/8` |
| Q: | `Q:<速度>` | 速度（BPM） | `Q:1/4=120` |
| K: | `K:<调号>` | 调号 | `K:D` `K:Am` `K:Bb` |

> **调号格式注意**：大调直接写字母（`K:D`），小调加 `m`（`K:Am`）。不要写 `K:Dmaj` 或 `K:Amin`。

### 音符语法

| 元素 | 写法 | 含义 |
|------|------|------|
| 基本音 | `C` `D` `E` `F` `G` `A` `B` | 白键音（大写 = 中央八度以下） |
| 升号 | `^C` `^F` | 升半音（前缀 `^`） |
| 降号 | `_B` `_E` | 降半音（前缀 `_`） |
| 高八度 | `c` `d` `e` | 小写 = 高一个八度 |
| 更高八度 | `c'` `d''` | 撇号 = 继续升高 |
| 低八度 | `C,` `D,,` | 逗号 = 降低八度（每逗一个） |
| 休止 | `z` | 休止符 |
| 时值 | `C2` `A/2` | 数字 = 乘（`C2` = 二分音符），`/` = 除（`A/2` = 八分音符） |
| 附点 | `C3/2` | 附点音符（原时值 ×1.5） |
| 连音 | `(3cde` | 三连音（`(3` 前缀） |

> **⚠️ Agent 必读：L:1/8 时值换算速查**
>
> 当 `L:1/8` 时，默认单位 = 八分音符 = **0.5 拍**（四分音符 = 1 拍）。
> 数字后缀表示「乘以该单位的几倍」。
>
> | ABC 写法 | 计算 | 实际拍数 | 音乐含义 |
> |----------|------|---------|---------|
> | `d` | 1 × 0.5 | 0.5 拍 | 八分音符 |
> | `d2` | 2 × 0.5 | 1 拍 | 四分音符 |
> | `d4` | 4 × 0.5 | 2 拍 | 二分音符 |
> | `d6` | 6 × 0.5 | 3 拍 | 附点二分音符 |
> | `d8` | 8 × 0.5 | 4 拍 | 全音符 |
> | `d/2` | 0.5 × 0.5 | 0.25 拍 | 十六分音符 |
>
> **4/4 拍一满小节 = 4 拍 = 8 个八分音符时值**
>
> 常见错误：误以为 `d4` = 4 拍。实际上 `d4` = **2 拍**（二分音符），`d8` 才是 4 拍。

### 和弦记谱

使用方括号将多个音组合为同时发声的和弦：

```
[CEG]     → C 大三和弦
[FAc]     → F 大三和弦（跨八度）
[_BDF]    → Bdim 减三和弦
[CEG2]    → 和弦持续二分音符时值
```

### 多声部（Voice）

使用 `V:` 声明独立声部，每个声部可指定不同谱号和 MIDI 参数：

```
V:1 name="Melody" clef=treble
V:2 name="Chords" clef=bass
V:3 name="Drums" clef=perc
```

### MIDI 指令

在声部声明后追加 MIDI 参数控制通道和音色：

> **Channel 编号说明**：ABC `%%MIDI channel` 使用 0-indexed（0-15），即 channel 9 = MIDI Channel 10（GM 标准鼓组通道）。下文所有 `channel 9` 均指 GM Channel 10。

```
V:1 name="Melody" clef=treble
%%MIDI channel 1
%%MIDI program 80        ← Square Lead

V:3 name="Drums" clef=perc
%%MIDI channel 9
%%MIDI program 0         ← 打击乐固定 0（GM Channel 10）
```

### 力度标记

嵌入在音符前或独立行使用：

| 标记 | 含义 | ABC 写法 |
|------|------|---------|
| pp | 极弱 | `!pp!` |
| p | 弱 | `!p!` |
| mp | 中弱 | `!mp!` |
| mf | 中强 | `!mf!` |
| f | 强 | `!f!` |
| ff | 极强 | `!ff!` |

### 小节线

| 符号 | 含义 |
|------|------|
| `\|` | 小节线 |
| `\|\|` | 双小节线（段落边界） |
| `\|:` | 反复开始 |
| `:\|` | 反复结束 |
| `\|1` / `\|2` | 第一/二结尾 |

### GM 鼓组映射（Rhythmist Agent 专用）

鼓组声部（`clef=perc`，ABC channel 9 = MIDI Channel 10）使用以下 ABC 音符对应 GM 打击乐：

| ABC 记谱 | 音色 | MIDI Note | 节奏功能 |
|----------|------|-----------|---------|
| `B,,` | Kick (低音鼓) | 36 | 主干，重拍 |
| `D,` | Snare (军鼓) | 38 | 主干，反拍 |
| `c` | Closed Hi-Hat (闭镲) | 42 | 主干，律动感 |
| `d` | Open Hi-Hat (开镲) | 46 | 点缀，节奏变化 |
| `F,` | Low Tom (低汤姆) | 45 | 填充，过渡 |
| `A,` | High Tom (高汤姆) | 50 | 填充，过渡 |
| `a` | Crash (吊镲) | 49 | 标记，段落开始/高潮 |
| `g` | Ride (叮叮镲) | 51 | 主干，稳定律动 |

> 鼓组声部的 `%%MIDI channel` 必须为 `9`（= MIDI Channel 10），`%%MIDI program` 必须为 `0`。

### 完整示例

```
X:1
T:Battle Theme
M:4/4
L:1/8
Q:1/4=140
K:D

V:1 name="Melody" clef=treble
%%MIDI channel 1
%%MIDI program 56
|: d2 f2 a2 f2 | a4 g2 f2 :|
|: d2 ^f2 a2 g2 | f4 d2 z2 :|

V:2 name="Chords" clef=bass
%%MIDI channel 2
%%MIDI program 48
| [DF,A,]4 [DFA]4 | [G,B,D]4 [G,B,d]4 |
| [DF,A,]4 [DFA]4 | [CEG]4 [CEG]4 |

V:3 name="Drums" clef=perc
%%MIDI channel 9
%%MIDI program 0
| B,,z D,z B,,z z2 | B,,z D,z B,,z D,z |
| B,,z D,z B,,z z2 | B,,z D,z a4 |
```

## 和弦构建参考

以根音 C (MIDI 60) 为例，其他根音按半音偏移。

### 三和弦与七和弦

| 和弦类型 | 音程 | C 和弦音 | 使用场景 |
|---------|------|---------|---------|
| 大三 (Major) | 0 4 7 | C E G | 通用 |
| 小三 (Minor) | 0 3 7 | C Eb G | 暗色、悲伤 |
| 属七 (Dom7) | 0 4 7 10 | C E G Bb | 紧张、解决 |
| 大七 (Maj7) | 0 4 7 11 | C E G B | 梦幻、爵士 |
| 小七 (Min7) | 0 3 7 10 | C Eb G Bb | 爵士、柔和 |
| 减三 (Dim) | 0 3 6 | C Eb Gb | 紧张、悬疑 |
| 减七 (Dim7) | 0 3 6 9 | C Eb Gb A | 紧张、过渡、恐怖 |
| 增三 (Aug) | 0 4 8 | C E G# | 悬浮、不安 |
| 挂四 (sus4) | 0 5 7 | C F G | 悬挂、期待（常解决到大三） |
| 挂二 (sus2) | 0 2 7 | C D G | 开放、空灵（可保持） |
| 属九 (9) | 0 4 7 10 14 | C E G Bb D | 爵士、放克 |
| 加九 (add9) | 0 4 7 14 | C E G D | 明亮、流行 |

### 转位表示法（详见术语表「罗马数字」）
- 第一转位：`C/E`（三音在低音）
- 第二转位：`C/G`（五音在低音）
- 用于声部连接时减少跳跃、避免低音重复。

### 挂留和弦解决倾向
- **sus4** → 大三和弦（三音下移半音）
- **sus2** → 大三或保持开放（可视为 add9 省略三音）

## 和弦排列法

### 密集排列 (Close Voicing)
- 和弦音在一个八度内紧密排列
- 适合：独奏乐器、轻音乐、Pad
- 示例 Dm：D4-F4-A4

### 开放排列 (Open Voicing)
- 和弦音跨越超过一个八度
- 适合：管弦乐编曲（铜管、弦乐组）
- 听起来更宏大、宽广、专业
- 生成方法：
  1. 取三和弦的三个音（如 D-F-A）
  2. 保留根音和五音在原位
  3. 将三音（F）降低一个八度（pitch - 12）
  4. 得到：F3-D4-A4

### 根音位置与转位
- **根音位置**：根音在最低声部，稳定性最高。
- **第一转位**：三音在最低声部，柔和，常用于连接。
- **第二转位**：五音在最低声部，常出现在终止式中（如 I 和弦的第二转位作装饰）。

### 四部和声基本规则（适用于弦乐组/合唱）
- 声部从上到下：Soprano（旋律），Alto，Tenor，Bass。
- 避免声部交叉（即 Alto 低于 Tenor 等）。
- 两个外声部（Soprano 与 Bass）应保持适当距离（通常不超过八度+八度）。
- 内声部保持流动但避免过多跳跃。

### 编曲排列规则
- **Strings 和 Brass 不要使用相同的和弦排列**
- Brass 倾向开放排列（更宽广、更有力量）
- Strings 可以使用密集排列（更温暖、融合）
- 如果多个乐器演奏相同和弦，至少让一个使用转位或开放排列
- 低音区（C2-B2）不要放密集和弦（会浑浊）
- 低音线与上方声部做反向运动可增加张力（例如旋律上行时，贝斯下行）

### 频率范围与复音限制

避免不同乐器在相同频率区域叠加导致浑浊：

| 频率区域 | 范围 | 建议同时声部数 | 主要乐器 |
|----------|------|---------------|----------|
| 低频 | C1-C2 | ≤ 2 | Bass、Kick Drum |
| 中低频 | C2-C3 | ≤ 3 | Bass、和弦根音、Tom |
| 中频 | C3-C5 | ≤ 4 | 和弦/Pad、旋律 |
| 高频 | C5-C7 | ≤ 6 | 旋律、装饰、Hi-Hat |

- 低频区（C2 以下）避免放密集和弦
- 旋律和低音不应在同一音区竞争

## 声部进行规则

和弦切换时的声部连接：

1. **共同音保持** — 两个和弦共有的音保持不动
2. **其他声部级进** — 非共同音尽量做半音或全音移动
3. **避免平行五八度** — 两个声部不要同时做纯五度或纯八度移动
4. **低音反向** — 低音线与上方声部做反向运动增加张力
5. **导音解决** — 导音（大调中 B，小调中 ^B）应级进上行到主音（C）
6. **增音程回避** — 在和声小调中，III 级与 VI 级之间的增二度不宜出现在同一旋律线中，可通过声部调整或使用自然小调替代

## 和弦外音

旋律中合理使用和弦外音可以增加流动感和张力，避免"每个音都是和弦音"的生硬感。

| 类型 | 定义 | 位置 | 效果 |
|------|------|------|------|
| 经过音 | 连接两个和弦音的级进非和弦音 | 弱拍 | 流动感、连接乐句 |
| 邻音 | 从和弦音级进后返回（上/下邻音） | 弱拍 | 装饰、丰富 |
| 延留音 | 前和弦的音延留到新和弦，再级进解决 | 强拍→弱拍 | 张力→释放，非常有表现力 |
| 先现音 | 下一个和弦的音提前出现 | 弱拍，时值短 | 推进感、衔接 |
| 倚音 | 强拍上的非和弦音，级进解决到和弦音 | 强拍 | 张力、情感表达 |

**使用原则：**
- 经过音和邻音可自由使用（已有乐句衔接规则覆盖）
- 延留音和倚音制造较强张力，每段使用 1-2 次即可，不宜过多
- 所有和弦外音最终应级进（半音或全音）解决到和弦音

## 和弦进行库

**罗马数字规则**（详见术语表）：大调和弦大写（I II III IV V VI VII），小调和弦小写（i ii iii iv v vi vii）。
**进行长度**：通常每个罗马数字代表 1 小节，可根据需要调整。

### 通用进行

| 进行 | 罗马数字 | 气质 | 常见场景 |
|------|---------|------|---------|
| 流行万能 | I - V - vi - IV | 温暖、积极 | 主菜单、村庄、教程 |
| 经典终止 | I - IV - V - I | 坚定、完整 | 通用 |
| 50年代 | I - vi - IV - V | 怀旧、甜蜜 | 休闲 |
| 悲伤流行 | vi - IV - I - V | 感伤、思念 | 离别、回忆 |
| 卡农 | I - V - vi - iii - IV - iii - ii - V | 优雅、深情 | 感人场景 |
| 爵士基础 | ii - V - I | 优雅、流动 | 爵士、休息区 |
| 摇滚 | I - IV - V | 力量、直接 | 战斗、动作 |

### 战斗/激烈进行

| 进行 | 罗马数字 | 气质 |
|------|---------|------|
| 力量感 | i - VII - VI - VII | 紧张、激烈 |
| 暗黑力量 | i - III - VII - VI | 黑暗、压迫 |
| 史诗战斗 | i - iv - VII - III | 史诗、壮阔 |
| 骑士出征 | i - VI - III - VII | 英雄、决意 |
| Boss战 | i - VII - VI - VII | 极端紧张（同力量感但力度更强） |

### 和平/探索进行

| 进行 | 罗马数字 | 气质 |
|------|---------|------|
| 宁静田园 | I - iii - IV - V | 平和、自然 |
| 温暖成长 | I - ii - iii - IV | 温暖、希望 |
| 森林探索 | I - V - vi - iii | 好奇、冒险 |
| 神秘探索 | i - iv - ii - V | 神秘、不安 |
| 梦幻 | I - ii - vi - V | 梦幻、漂浮 |

### 恐怖/暗黑进行

| 进行 | 罗马数字 | 气质 |
|------|---------|------|
| 悬疑 | i - iv - V - i | 神秘、不安（改良） |
| 暗黑悬疑 | i - II - VII - i | 黑暗、诡异 |
| 恐怖 | i - bII - i - bIII | 极端恐怖 |
| 压迫 | i - VII - i - VI | 压迫、绝望 |

> 注：原 `i-iv-ii-V` 与神秘探索重复，此处改为 `i-iv-V-i` 增强悬疑终止感。

### 情感/戏剧进行

| 进行 | 罗马数字 | 气质 |
|------|---------|------|
| 深情 | I - V - vi - IV - I - iii - IV - V | 深情、动人 |
| 史诗 | I - V - vi - IV - iii - IV - I - V | 史诗、壮阔 |
| 悲壮 | i - iv - VII - III - iv - VII - i | 悲壮、英雄 |
| 希望 | vi - IV - I - V | 希望、重生 |

### 终止式库（详见术语表）

> **注意：终止式仅用于段落结尾（A→B 过渡、B 段结束等），不要在每个乐句结尾都使用终止式！** 乐句内部应保持流动，结尾可停在经过音或非和弦音上制造悬念。

| 名称 | 进行 | 效果 | 使用场景 |
|------|------|------|----------|
| 正格终止 | V - I | 最稳定，结束感强 | 段落/全曲结尾 |
| 变格终止 | IV - I | 柔和，宗教感 | 段落结尾 |
| 半终止 | 结束于 V | 未完待续 | A 段结尾（过渡到 B 段） |
| 阻碍终止 | V - vi (或 V - VI) | 意外，延宕 | 需要延长段落时 |
| 假终止 | V - bVII | 转向其他调性，戏剧性 | 剧情转折点 |

### 离调与转调

游戏音乐中常通过临时离调来制造对比和新鲜感。

**常用离调方法：**

| 方法 | 操作 | 效果 | 示例（C 大调） |
|------|------|------|---------------|
| 属调离调 | 临时借用 V 级的 V 和弦 | 明亮、推进 | C → D（V/V）→ G → C |
| 下属调离调 | 临时借用 IV 级的 I 和弦 | 柔和、开阔 | C → F（IV）→ d（iv）→ C |
| 关系小调交替 | 大调↔小调自由切换 | 情感色彩变化 | C → a（vi）→ F → C |
| 平行调交替 | 同主音大小调混合 | 戏剧性 | C → c（i）→ G → C |
| 借用和弦 | 从平行调借用和弦 | 增加色彩 | C 大调借用 `iv`（Fm）或 `bVII`（Bb）|

**使用原则：**
- **B 段是最佳离调位置**：A 段建立主调，B 段离调制造对比，回到 A 段时回归主调
- 离调通常持续 2-4 小节，不要超过半首曲子
- 离调结束后必须回到主调（通过终止式或共同和弦过渡）
- 短曲（<30 秒）建议**不转调**，用离调即可
- 长曲（>60 秒）可以在 C 段或 D 段做真正的转调

## 旋律音程指南

> 以下示例均以 **C 大调** 为参考调。音程关系在其他调性中相同，只需替换音名。

### 度数与半音对应表

以下以 C 大调为例，列出音程与 ABC 音名的对应关系：

| 音程 | ABC 示例 | 半音数 | 分类 |
|------|---------|--------|------|
| 纯一度 | C→C | 0 | 同音 |
| 小二度 | E→F, B→c | 1 | 级进 |
| 大二度 | C→D, D→E, F→G, G→A, A→B | 2 | 级进 |
| 小三度 | D→F, E→G, A→c, B→d | 3 | 小跳 |
| 大三度 | C→E, F→A, G→B | 4 | 小跳 |
| 纯四度 | C→F, D→G, E→A, G→c, A→d, B→e | 5 | 常用跳进 |
| 三全音 | F→B, B→F | 6 | 需谨慎 |
| 纯五度 | C→G, D→A, E→B, F→c, G→d, A→e | 7 | 常用跳进 |
| 小六度 | E→c, A→f, B→g | 8 | 大跳 |
| 大六度 | C→A, D→B, F→d, G→e | 9 | 大跳 |
| 小七度 | D→c, E→d, G→f, A→g, B→a | 10 | 大跳 |
| 大七度 | C→B, F→e | 11 | 需特殊处理 |
| 纯八度 | C→c | 12 | 大跳 |

> 小调中 D→E 为大二度(2半音)、E→F 为小三度(3半音)，需注意与上述大调表的差异。

### 音程使用频率分级

**极常用（级进）：**
- 大二度(2)、小二度(1) — 旋律运动的基础，任何时候都可以使用
- 小三度(3)、大三度(4) — 最常见的跳跃，自然流畅

**常用（小跳）：**
- 纯四度(5)、纯五度(7) — 标志性跳进，JRPG 旋律大量使用
- 特别适合段落开头和乐句重音

**需谨慎（大跳）：**
- 大六度(9)、小七度(10) — 可产生戏剧效果，但需要处理
- 大七度(11) — 极少直接使用，除非有特殊音乐意图

**需特殊处理：**
- 三全音/增四度/减五度(6) — 不协和音程，必须正确解决
- 增二度(如和声小调 6→#7 = 3半音) — 避免在旋律中使用

### 大跳处理规则

当旋律中出现 5 半音以上的跳进时：

1. **反向级进收回**（最重要）：大跳后立即反向级进 1-2 步回到正常音区
   - 好：`C → G → F → E`（上跳后级进下行）
   - 差：`C → G → B → d`（连续同向大跳）
2. **不超过连续 2 次同向大跳**
3. **大跳落在和弦音上**：跳进的目标音应是当前和弦的构成音
4. **例外**：段落开头允许一个大跳作为"起句"，但必须立即反向收回

### 音域约束

- **推荐音域**：1.5 个八度（18 半音），从中央八度向上下各扩展约 9 半音
- **高潮段扩展**：可临时扩展到 2 个八度（24 半音），但不宜持续
- **音域标记**：使用逗号 `,`（低八度）和撇号 `'`（高八度）标记，避免超过 2 个八度标记
- **音区过渡**：需要从低音区到高音区时，通过级进逐步过渡，不要突然跳转

### 增减音程处理

- **增四度/减五度**（三全音）：必须解决。向上进行时解决到上方大二度（如 `F→B→c`），向下进行时解决到下方大二度（如 `B→F→E`）
- **增二度**（和声小调 `A→B`）：在旋律中回避。使用自然小调代替（`A→_B`），或用其他音代替升号音
- **经过性使用**：增减音程可以作为经过音短暂出现，但必须在下一拍解决到协和音程

## 旋律写作技法

### 动机（Motif，详见术语表）

动机是旋律的最小构成单位，通常 2-4 个音符，具有辨识度和记忆点。一首好曲子的旋律往往围绕 1-2 个核心动机展开。

**动机类型：**
- **音高动机**：由特定的音程关系定义（如 `C-E-G` 上行三度跳进）
- **节奏动机**：由特定的节奏型定义（如 `♩ ♪♪ ♩` 长短短长）
- **混合动机**：音高 + 节奏的组合（如 `♩ C ♪ E ♪ G`）

### 动机发展 6 法

1. **重复**：完全重复动机，建立记忆。适合段落开头或副歌
   - `C E G | C E G`
2. **变奏**：改变动机的某个要素（音高、节奏、力度），保留核心特征
   - 音高变奏：`C E G | C E A`
   - 节奏变奏：`C E G - | C E G. A`
   - 力度变奏：`(mf) C E G | (f) C E G`
3. **模进**：在不同音高上重复动机（上行/下行），制造推进感
   - `C D E | D E F | E F G`
4. **倒影**：上下翻转动机的音程方向
   - 原型 `C E G` → 倒影 `C _A _G`（音程方向反转）
5. **扩展**：在动机基础上添加音符，延长或丰富
   - 原型 `C E G` → 扩展 `C D E G` 或 `C E F G`
6. **截断**：取动机的一部分，制造收束感或变化
   - 原型 `C E G A` → 截断 `C E G` 或 `E G A`

### 变奏技巧细化

- **装饰音变奏**：在原动机音符前添加邻音（如 `C E G` → `C D E G`）
- **节奏拉伸/压缩**：将动机的时值整体拉伸（更舒缓）或压缩（更紧凑）
- **音区移位**：将动机移到高八度或低八度重复，制造呼应效果
- **调式交替**：在大调和小调之间切换同一动机（如 `C D E` ↔ `C D _E`）

### 乐句构建

**乐句长度：**
- 2 小节（短句）：适合快速主题、游戏循环短曲
- 4 小节（标准句）：最常见，对称感强
- 8 小节（长句）：适合展开段落、抒情旋律

**前句/后句关系：**
- **问答式**：前句结束在不稳定音（如 D, G, B），后句解决到稳定音（C, E）
  - 前句：`C D E G | D - - -`
  - 后句：`E F G A | G E C -`
- **平行式**：前后句使用相同动机，不同结尾
  - 前句：`C D E G | A - - -`
  - 后句：`C D E G | C - - -`

**弱起技巧：**
新乐句从弱拍进入（如第 4 拍或第 2 拍），而非每个乐句都从强拍正拍起。弱起增加流动感，避免"军队式"的刻板节奏。

### 旋律建筑

**音域布局：**
- 低→高→低（拱形）：最自然的旋律走向，适合大多数风格
- 阶梯式上升：逐步爬升到高潮，适合战斗/Boss 音乐
- 波浪式：在高音区和低音区之间交替，适合探索/村庄音乐

**高潮位置：**
- 黄金分割点：全曲 60-80% 位置设置最高音
- 高潮前的准备：高潮前 4-8 拍逐步升温（音区上升、力度增强、节奏加密）
- 高潮后的回落：高潮后不要立即回到最低音区，用 4-8 拍逐步回落

**段落对比（A 段与 B 段必须有明显区别）：**
- 旋律：不同的音域、节奏、动机
- 和声：不同的和弦进行
- 配器：增减乐器数量
- 力度：不同的 velocity 范围
- 节奏：不同的鼓点密度

### 旋律与和弦配合

**强拍（小节第 1 拍）：** 旋律音应落在当前和弦的构成音上
- 当前和弦为 C（C-E-G）时，强拍使用 C、E、G
- 当前和弦为 Am（A-C-E）时，强拍使用 A、C、E

**弱拍（小节第 2-4 拍）：** 可以使用和弦外音，增加色彩
- 经过音：在两个和弦音之间级进经过（如和弦音 C→E 之间加 D）
- 邻音：从和弦音级进到相邻的非和弦音再回来（如 `G A G`）
- 倚音：弱拍使用非和弦音，下一拍解决到和弦音（如 `B → C`）

**避免：**
- 不要每个乐句结尾都落在和弦音上（产生过多终止感）
- 不要强拍也用和弦外音（听起来不协调）
- 终止式只在段落结尾使用（详见终止式库）

## 节奏声部

### 鼓组节奏模式库

以下模式均基于 4/4 拍、`L:1/8`（每个 `x` = 八分音符）。一个模式 = 1 小节。

**基本战斗鼓型（通用）：**
```
B,, z D, z B,, z D, z     ← 标准四四拍
B,, z D, z B,, B,, D, z   ← 第四拍底鼓加速（段落结尾 fill）
```

**摇滚/现代 JRPG 鼓型：**
```
B,, z D, z B,, z D, D,     ← 反拍军鼓变化
B,, B,, D, z B,, z z D,    ← 密集底鼓推进
```

**电子/科幻鼓型：**
```
B,, z D, z B,, z D, c c    ← 末尾闭镲加速
B,, B,, D, z B,, z D, z    ← 十六分底鼓驱动（L:1/16 时）
```

**轻柔探索/村庄鼓型：**
```
z z D, z z z D, z          ← 仅军鼓 + 休止
c z D, z c z z z           ← 闭镲为主，极简
```

**Boss 战高潮鼓型：**
```
B,, B,, D, D, B,, B,, D, D,  ← 全部四分音符
B,, z D, D, B,, B,, D, z    ← 三连军鼓 fill
```

**Fill（填充/过渡）模式（放在段落最后 1 小节）：**
```
B,, z D, D, D, D, D, D,    ← 军鼓连打（经典 fill）
D, D, D, D, D, D, D, D,    ← 纯军鼓连打（力度递增）
F, z A, z a z z z z        ← 汤姆→吊镲 fill
```

> **使用原则**：普通段落用基本鼓型或风格鼓型，段落结尾最后 1 小节用 Fill，高潮段用 Boss 鼓型。不要每小节都变化鼓型，一个段落内保持 1-2 种模式。

### 低音线节奏模式

| 模式 | 节奏 | 适用风格 |
|------|------|----------|
| 持续低音 | 全音符(4.0)或二分音符(2.0) | 氛围、慢板 |
| 行走低音 | 四分音符(1.0)连续，根音→经过音→和弦音 | 爵士、放克 |
| 八分驱动 | 八分音符(0.5)，根音-五音交替 | 摇滚、电子 |
| 切分低音 | 弱拍(&/反拍)强调重音 | 放克、R&B |
| 琶音低音 | 十六分音符(0.25)分解和弦，根音-三音-五音-八度 | 电子、史诗 |

### 低音线音高选择规则

**基本原则：**
1. **强拍（第 1、3 拍）弹根音**：和弦转换时，强拍必须明确标示新的根音
2. **弱拍填充**：在根音之间插入和弦音或经过音，保持运动感
3. **音域范围**：E2-B3（MIDI 40-59），不要超过此范围

**常见模式（以 C 大调 I-V-vi-IV 为例）：**

| 模式 | I (C) | V (G) | vi (Am) | IV (F) | 适用风格 |
|------|-------|-------|---------|--------|---------|
| 根音-五音 | C-G | G-D | A-E | F-C | 摇滚、通用 |
| 根音-经过 | C-D-E | G-A-B | A-G-F | F-E-D | 爵士、行走 |
| 根音-八度跳 | C-c | G-g | A-a | F-f | 放克、电子 |
| 纯根音 | C | G | A | F | 氛围、极简 |

**连接技巧：**
- 和弦切换时，用**经过音**连接前后根音（如 G→A 之间插入 `^F` 或 `G`）
- 避免连续两个根音之间超过纯五度的大跳（超过时用经过音填补）
- 低音线与旋律做**反向运动**可增加张力（旋律上行时低音下行）

## 乐器演奏约束

> **适用范围**：以下约束适用于传统管弦乐/原声配器场景。电子音乐、合成器、特殊音效等风格可酌情突破（如合成器不受换气限制，方波/锯齿波不受演奏技法限制），但突破时应有明确的音乐意图。

### 弦乐组（小提琴、中提琴、大提琴）

| Do | Don't | 原因 |
|----|-------|------|
| 多用连音 `(CDE F)` 表达乐句 | 连续同音无连线（"锯木"感） | 连音线使 legato 自然 |
| 长音（≥2 拍）标注颤音 `!tr!` | 长音平直无变化 | 弦乐长音自然带颤音 |
| 拨奏用短音符（8/16 分） | 快速经过句中大量拨奏 | 拨奏过快难以清晰 |
| 双音跳进不超过六度 | 连续双音大跳 | 需换把，难度高 |

### 木管组（长笛、双簧管、单簧管）

| Do | Don't | 原因 |
|----|-------|------|
| 利用音区特性（低音温暖、高音尖锐） | 长笛持续极低音（<C4） | 低音区气声重、穿透力弱 |
| 装饰音用 `{g}f` 或 `!trill!` | 极高音区密集装饰音 | 高音区指法复杂 |
| 乐句间留出换气时间（休止或长音尾） | 密集无停顿超过 8 小节 | 木管需要换气 |
| 断奏 `!staccato!` 适合颗粒感段落 | 整个乐句全用断奏 | 破坏连贯性 |

### 铜管组（小号、圆号、长号）

| Do | Don't | 原因 |
|----|-------|------|
| 强音配合 pitch bend 作为音头 | 快速段落连续大跳（>八度） | 需改变口型，易破音 |
| 休止至少 1 拍用于换气 | 音符密集无停顿 | 铜管需大量气息 |
| 弱音器 `+con sordino+` 用于特殊色彩 | 长时间高音强奏（>4 小节） | 嘴唇负担极大 |
| 圆号阻塞音 `+stopped+` 制造金属感 | 圆号高音区（>G5）长时间持续 | 控制难度大 |

### 鼓组（ABC channel 9 = MIDI Channel 10）

| Do | Don't | 原因 |
|----|-------|------|
| 滚奏用 `!roll!` 标注 | 用 32 分音符手写滚奏 | 冗余，滚奏由演奏者决定密度 |
| 吊镲 `a` 用 `!>!` 强调重音 | 开镲 `d` 每拍正拍使用 | 高频掩蔽其他乐器 |
| 底鼓和军鼓交替律动 | 两者完全同步 | 同步削弱节奏层次感 |
| 段落变化时调整鼓 pattern | 连续 4+ 小节相同 pattern | 缺乏推进感 |

### 键盘与合成器

| Do | Don't | 原因 |
|----|-------|------|
| 和声用开放排列 `[FAc]` 跨八度 | 密集排列覆盖旋律音区 | 开放排列留出旋律空间 |
| 钢琴踏板 `!ped!` / `!pedend!` 标记 | 快速和声变换处持续踏板 | 和声混浊 |
| Pad 配合 CC11 渐变 | 所有和弦相同力度 | 缺乏呼吸感 |
| 声像分离（CC10）增强空间感 | 所有声部相同声像 | 扁平无深度 |

### ABC 演奏标记速查

| 标记 | 效果 | 适用乐器 |
|------|------|---------|
| `(CDE F)` | 连音线（legato） | 弦乐、木管 |
| `!staccato!` | 断奏（缩短音符） | 木管、弦乐 |
| `!tr!` / `{g}f` | 颤音/装饰音 | 全部旋律乐器 |
| `!slur!` | 滑音标记 | 弦乐、铜管 |
| `!roll!` | 滚奏 | 鼓组 |
| `!>!` | 强调重音 | 鼓组（吊镲） |
| `!ped!` / `!pedend!` | 踏板踩下/释放 | 钢琴 |
| `+pizz+` | 拨奏 | 弦乐 |
| `+con sordino+` | 弱音器 | 铜管 |
| `+stopped+` | 阻塞音 | 圆号 |

## 配器平衡原则

### 频段分配与乐器数限制

| 频段 | 频率范围 (Hz) | MIDI 范围 (约) | 主导乐器 | 同时演奏上限 |
|------|-------------|---------------|---------|------------|
| 次低频 | 20–60 | C0–B1 | 底鼓、Sub Bass | 1–2 件 |
| 低频 | 60–200 | C2–G3 | 贝斯、大提琴、低音铜管 | ≤2 件 |
| 中低频 | 200–500 | G3–B4 | 吉他、钢琴左手、中提琴、圆号 | ≤3–4 件 |
| 中频 | 500–2000 | C5–B6 | 人声、小提琴、长笛、铜管 | 留空间给旋律 |
| 中高频 | 2k–6k | C7+ | 镲片、木管高音、弦乐泛音 | 控制刺耳感 |
| 高频 | 6k–20k | — | 镲片泛音、空气感 | 控制能量 |

**关键规则：**
- 低频区（C2 以下）只保留 1–2 件乐器（如贝斯 + 底鼓），和弦根音使用开放排列
- 中低频区（G3–B4）是"泥泞区"，最易浑浊，非核心乐器应减少或 EQ 衰减
- 旋律乐器应突出在中高频（2k–5k），避免被和声掩蔽
- 参考上文「频率范围与复音限制」表格的详细分频段复音上限

### 配器层次（从前到后）

| 层次 | 乐器类型 | CC91 混响 | CC7 音量 | 说明 |
|------|---------|----------|---------|------|
| 最前层 | 旋律（独奏/齐奏） | 少（40-60） | 最高（95-110） | 保持靠前、清晰 |
| 前层 | 对位旋律、装饰 | 中（60-75） | 中高（80-95） | 辅助旋律，不抢主位 |
| 中层 | 和声、Pad | 较多（70-85） | 中（70-85） | 拉远距离，铺底 |
| 后层 | 低音、节奏 | 短混响（50-65） | 中（75-90） | 保持冲击力 |

### 声像定位（CC10）

| 乐器类型 | CC10 值（0=左, 64=中, 127=右） |
|---------|-----------------------------|
| 旋律（主奏） | 64（中央） |
| 低音（贝斯） | 64（中央） |
| 底鼓 | 64（中央） |
| 和声/Pad | 44–50（略偏左）或 78–84（略偏右） |
| 钢琴左手/右手 | 44 / 84 |
| 小提琴 | 40（偏左） |
| 大提琴 | 88（偏右） |
| 中提琴 | 64（中央） |
| 铜管组 | 40–50 或 78–88 |
| 军鼓 | 58（略偏左） |
| 镲片 | 左右散开（30 / 98） |

**原则：** 低音和主旋律保持中央；其他乐器对称摆位；CC10 变化不宜频繁。

### 常见平衡问题诊断

| 问题 | 症状 | 根因 | 修正方法 |
|------|------|------|---------|
| 浑浊 | 整体听感模糊，缺乏清晰度 | 中低频乐器过多或重叠 | 减少中低频乐器数，或移高八度，或使用开放排列 |
| 旋律被掩蔽 | 主旋律听不到或被淹没 | 和声/节奏与旋律同音区且音量相当 | 将和声音区降低/升高，减弱和声力度（CC7-10） |
| 低频不足 | 缺乏根基感，听感单薄 | 贝斯或底鼓缺失/过弱 | 确保至少一件乐器覆盖 60–120 Hz（约 C2–B2） |
| 高频刺耳 | 镲片或合成器让人不适 | 高频乐器音量过高或密度过大 | 降低音量，减少高频乐器使用密度 |
| 左右失衡 | 一侧过重，头部分离感 | 重要乐器偏侧 | 主旋律、贝斯放中央，其他乐器对称摆位 |
| 动态平淡 | 缺乏起伏，机械感 | 所有段落同一力度 | 在段落边界加入渐强/渐弱（CC11 渐变 4-8 拍） |

### 音色搭配原则

1. **互补原则** — 明亮乐器（长笛、小号）与温暖乐器（弦乐、圆号）搭配增加层次
2. **避免同质化** — 不使用多件音色相似乐器演奏相同旋律（如三把小提琴齐奏），用不同八度或装饰音区分
3. **同和弦多乐器** — 至少一件使用开放排列或转位，避免密集排列重叠
4. **乐器特性** — 长笛高音穿透力强适合旋律，低音区柔弱不宜支撑；圆号中音区温暖适合和声，高音区易"炸"需控力度

### 风格旋律特征

每种游戏音乐风格有其独特的旋律特征，Composer 应根据风格选择合适的旋律写法。

> **灵活运用原则**：以下特征是各风格的典型倾向，不是硬性规则。实际创作中可根据具体场景、主题和创意需要灵活调整——例如在悲伤曲调中融入希望动机，或在战斗音乐中加入抒情段落，都是有效的创作手法。

#### 经典 JRPG 战斗

- **音域**: 中高音区为主（中央八度到高八度）
- **常用音程**: 纯四度、纯五度跳进，偶尔大六度
- **节奏**: 四分/八分音符为主，切分音点缀
- **动机**: 上行模进发展，铜管式号角动机（如 `C E G - | D F A -`）
- **特征**: 级进为主，偶有大跳但立即收回，旋律线条清晰有力

#### 现代 JRPG 战斗

- **音域**: 全音域覆盖，高低交替
- **常用音程**: 大跳 + 切分，小二度碰撞
- **节奏**: 密集，十六分音符跑动，切分重音
- **动机**: 电吉他式推弦动机（如 `C ^D E G`），快速音阶上行
- **特征**: 大跳多，节奏驱动强，旋律更"锋利"

#### RPG Boss

- **音域**: 全音域，从低到高递进
- **常用音程**: 纯五度、大六度、八度
- **节奏**: 长音 + 短音交替，附点节奏
- **动机**: 铜管号角 + 模进递进（如 `C - E - | G - c -`），高潮持续长音
- **特征**: 庄严、压迫感，模进层层递进到高潮

#### 8-bit / Chiptune

- **音域**: 窄（1 个八度内为主）
- **常用音程**: 小三度、纯四度
- **节奏**: 重复性强，等时值音符
- **动机**: 短小精悍的 2-3 音动机，大量重复
- **特征**: 有限音域内的跳跃式旋律，方波音色下的简洁线条

#### 史诗管弦

- **音域**: 宽广（2 个八度）
- **常用音程**: 纯五度、大六度，抒情时用级进
- **节奏**: 长线条，二分/全音符，渐强式乐句
- **动机**: 弦乐式连奏旋律，大跳后级进展开
- **特征**: 长音多，休止符制造呼吸，渐强渐弱明显

#### 暗黑 / 恐怖

- **音域**: 低音区为主（中央八度以下）
- **常用音程**: 减五度（三全音）、小二度、增二度
- **节奏**: 不规则，短动机 + 长休止
- **动机**: 不规则的短动机重复（如 `C _E D - | z _A G -`）
- **特征**: 小音域低区徘徊，减音程营造不安，动机碎片化

#### 悲伤 / 抒情

- **音域**: 中音区，下行旋律多
- **常用音程**: 小三度、大二度级进
- **节奏**: 慢速，附点、延音多
- **动机**: 下行叹息式动机（如 `G E D C | A G E -`），装饰音点缀
- **特征**: 级进下行，缓慢，装饰音（颤音/波音暗示）

#### 村庄 / 探索

- **音域**: 中音区，窄音域
- **常用音程**: 纯五度、小三度，五声音阶倾向（C D E G A）
- **节奏**: 轻松，四分音符为主，偶有八分音符
- **动机**: 简单 2-3 音动机，重复 + 微变（如 `E G A G | E G E -`）
- **特征**: 五声音阶感，简单温馨，休止多（呼吸感）

#### 主菜单

- **音域**: 宽广（1.5-2 个八度）
- **常用音程**: 大跳点缀（纯五度、大六度），级进为主
- **节奏**: 长音 + 短音交替，自由节奏感
- **动机**: 宽广旋律线，大跳后缓慢收回（如 `C - G - | E D C -`）
- **特征**: 抒情大气，不像战斗那样紧凑，有"展开"感

#### 科幻 / 电子

- **音域**: 中高音区
- **常用音程**: 小二度、增四度（金属感）
- **节奏**: 电子节拍驱动，十六分音符重复模式
- **动机**: 短小重复音型（如 `C C ^F - | C C _B -`）
- **特征**: 冷调，重复性强，音程不协和

#### 胜利

- **音域**: 高音区为主，明亮
- **常用音程**: 纯四度、纯五度上行，大六度
- **节奏**: 进行曲式，强拍重音
- **动机**: 上行号角式（如 `C E G c | G E C -`），模进上行
- **特征**: 昂扬向上，节奏坚定，铜管感

#### 沙漠 / 中东

- **音域**: 中音区
- **常用音程**: 增二度（特征音程）、小二度
- **节奏**: 附点节奏，切分
- **动机**: 装饰音多，邻音围绕主音（如 `C D E D C _B C -`）
- **特征**: 异域调式，增二度营造中东感，装饰音丰富

## 弯音（Pitch Bend）

### GM 标准弯音值

MIDI Pitch Bend 使用 14-bit 值（0-16383），8192 为居中（无弯音）。

GM 标准默认 Pitch Bend Range 为 **±2 半音**：

| 值 | 效果 | 与居中的距离 |
|----|------|-------------|
| 16383 | +2 半音（最大） | +8191 |
| 12288 | +1 半音 | +4096 |
| 8192 | 居中（无弯音） | 0 |
| 4096 | -1 半音 | -4096 |
| 0 | -2 半音（最小） | -8191 |

> **重要：** 实际弯音的音高变化范围取决于目标合成器的 **Pitch Bend Range** 设置。GM 默认为 ±2 半音，但 SF2 音色可能配置不同的范围（如 ±1 半音、±12 半音等）。
> **建议**：在发送弯音前，可通过 RPN 0（音高变化范围）设置所需范围，或确认音源默认值。

### 使用规则
- 弯音事件必须**成对使用**：弯上去 → 回中（8192），避免音高永久偏移
- **弯音斜率**：快速滑动（短时间）用于音头装饰（如吉他推弦），慢速弯音用于旋律表情（如小提琴滑音）
- 在 Godot Clef 播放器中，弯音值直接传递给 SF2 合成器

## 表现力注入参考（Orchestrator Agent 专用）

### 双层表现力机制

Clef 的表现力由两层叠加实现，两者不冲突：

1. **基础层（ABC 力度标记）**：`!pp!` `!mp!` `!f!` `!ff!` 等，由 abc_to_midi.py 转换为**音符 velocity**（0-127）。设定声部的基准动态级别。
2. **微调层（CC11 Expression）**：通过 expression_plan.json 注入到 MIDI，在 velocity 基准上做**实时力度微调**。

**协作规则**：
- ABC 力度标记设定段落级别的动态基线（如 A 段 `!mf!`，B 段 `!f!`，C 段 `!ff!`）
- CC11 在基线上做乐句级起伏（如 `!f!` 基线 100 上，CC11 在 80-120 之间波动）
- 最终音量 ≈ velocity × (CC11/127)，两者共同决定听感力度
- **不要只依赖一层**：如果只写 ABC 力度标记，动态变化会很生硬；如果只写 CC11 而所有音符 velocity 相同，基线不清晰

### CC 消息一览

| CC# | 名称 | 范围 | 效果 | 常用场景 |
|-----|------|------|------|---------|
| CC1 | Modulation (颤音深度) | 0-127 | 增加颤音/vibrato | 弦乐长音、铜管 |
| CC7 | Volume (通道音量) | 0-127 | 控制整体音量 | 段落音量变化 |
| CC10 | Pan (声像) | 0-127 | 左右定位 | 立体声宽度 |
| CC11 | Expression (表情) | 0-127 | 实时力度微调 | 渐强渐弱（比 CC7 更细腻） |
| CC64 | Sustain (延音踏板) | 0/63/127 | 延音开/关 | 钢琴、竖琴 |
| CC91 | Reverb (混响深度) | 0-127 | 空间感 | 氛围段落 |
| CC93 | Chorus (合唱深度) | 0-127 | 厚度 | 弦乐加厚 |

### 渐强渐弱实现

- **渐强**：从低值逐步递增 CC11（如 60→90→110→127）
- **渐弱**：从高值逐步递减 CC11（如 127→100→70→40）
- **突强 (sfz)**：音符 velocity 突然提高（如 60→120），常用于段落开头或重音
- **渐慢/渐快**：通过逐步拉长/压缩音符时值实现（ABC 中调整时值数字）

### 使用原则

1. **长音加颤音**：超过 1 拍的音符可加 CC1（起始 0，逐步升至 30-50）
2. **段落渐强**：高潮前 4-8 小节 CC11 从 70 逐步升至 127
3. **段落渐弱**：高潮后 CC11 从 127 逐步降至 50-60
4. **钢琴延音**：CC64=127 踩下，和弦切换前 CC64=0 释放
5. **不要同时大幅改变多个 CC**：一次最多改变 1-2 个 CC 参数

## 歌曲形式参考

### 基本结构
简单二元 (A-B): 段落A(2-4小节) → 段落B(2-4小节)
三段体 (A-B-A): A(2小节) → B(2小节) → A'(2小节)，经典对称
通谱体 (Through): 持续发展不重复，适合短曲
Rondo (A-B-A-C-A): 主部A与插部交替，适合游戏循环
史诗 (A-B-C-B-D): 多段递进，适合 Boss 战
渐强 (A-A'-A''-B): 逐步加强，A 每次变化，B 为高潮


### 乐句结构

#### 常见乐句长度

| 长度 | 特点 | 适用场景 |
|------|------|---------|
| 2 小节 | 短促有力，快速建立和解决 | 游戏循环短曲、重复性主题 |
| 4 小节 | 标准句长，对称感强，最常用 | 大多数游戏音乐、AB 对句 |
| 8 小节 | 长句，展开空间大 | 抒情旋律、发展段落 |

游戏循环通常使用 **2、4、8 小节的对称结构**，便于无缝重复。

#### 乐句关系类型

**平行**：前后句使用相同动机，仅结尾不同。建立统一感。
- 前句：`C D E G | A - - -`（停在 A，不稳定）
- 后句：`C D E G | C - - -`（停在 C，解决）

**对比**：前后句使用不同的动机材料。建立紧张感。
- 前句：`C D E G | A G E -`
- 后句：`E F G A | G F D -`

**问答式**：前句结束在不稳定音（D, G, B），后句解决到稳定音（C, E）。最自然的听感。
- 前句：`C D E G | D - - -`（停在 D，"问"）
- 后句：`E F G A | G E C -`（停在 C，"答"）

#### 乐句连接技巧

乐句之间应保持流动，避免过多的终止感。

- **抢先进入（Anticipation）**：下一乐句提前半拍或一拍开始
  - `... E G | A c ...`（第二乐句从第一乐句末尾开始）
- **弱起**：新乐句从弱拍（第 2、3、4 拍）进入，避免每个乐句都从强拍正拍起
  - `... G E | D F E G A ...`（新乐句从第二拍进入）
- **经过音连接**：用经过音或邻音连接两个乐句，而非用休止断开
  - `... G - | G A B c ...`（用 G-A-B 作为连接桥梁）
- **段落间停顿**：段落之间可以有明显的气息停顿（休止或长音），增强呼吸感和结构划分。与乐句间不同，段落间的停顿是合理的。

#### 终止式类型

终止式决定乐句/段落的结束感。**完整终止式列表详见上方「和弦进行库 → 终止式库」章节。**

> **重要**：终止式只在段落结尾使用。乐句中间不要使用终止式，否则旋律碎片化。

#### 段落结构

| 结构 | 形式 | 适用场景 |
|------|------|---------|
| 二段式 | A-B | 大多数游戏音乐 |
| 三段式 | A-B-A | 对称结构，A 段回归 |
| 通谱体 | 持续发展不重复 | 短曲、过场 |
| 回旋曲式 | A-B-A-C-A | 游戏循环，主部重复 |
| 史诗结构 | A-B-C-B-D | Boss 战，多段递进 |

### 段落对比手段（A 段与 B 段必须有明显区别）
- 旋律：不同的音域、节奏、动机
- 和声：不同的和弦进行
- 配器：增减乐器数量
- 力度：不同的 velocity 范围
- 节奏：不同的鼓点密度

### 游戏循环适配
- **探索/村庄**：多用 Rondo 或三元体，主部 A 稳定，插部稍作变化。
- **战斗/Boss**：多用通谱体或史诗结构，持续递进，高潮处使用全编制。
- **过场**：常用二元或三元体，简单清晰。

### 重复与变奏规则
- 同一段落在第二次出现时建议做 **配器加厚**（增加弦乐层、Pad）或 **旋律装饰**（加花、变奏）。
- 避免机械重复超过 2 次，否则容易疲劳。

### 段落过渡技巧

段落之间的过渡影响整体流畅感，应根据风格选择合适的过渡方式：

| 类型 | 方法 | 适用场景 |
|------|------|----------|
| 渐弱过渡 | A 段结尾 velocity 渐弱到 50，CC11 降到 40；B 段开头从弱渐强 | 氛围→高潮、探索→战斗 |
| 填充过渡 | 鼓组 fills：军鼓连打（最后 1 拍 4 个十六分音符 velocity 递增）、底鼓加速 | 战斗、摇滚、电子 |
| 悬留过渡 | A 段结尾使用属和弦 V 或减七和弦，B 段开头解决到主和弦 I | 史诗、古典、戏剧性 |
| 静默过渡 | A 段结尾突然休止 1-2 拍，B 段全奏进入 | 紧张→爆发、战斗开始 |
| 交叉渐变 | A 段配器逐渐淡出，B 段新配器同时淡入（velocity + CC11 交叉） | 平滑转换、氛围音乐 |

### 循环衔接技巧

游戏音乐需要无缝循环，以下技巧确保循环点（结尾→开头）不突兀。

**旋律衔接：**
- 循环点前 2-4 拍的旋律应**自然过渡**到循环开头的音区，避免结尾在高音区、开头在低音区的断层
- 如果 A 段开头是强拍起音，结尾也应落在强拍音上（而非弱拍）

**和声衔接：**
- 最后 1 小节使用**主和弦 I** 或**属和弦 V**，确保回到开头时和声顺畅
- 避免：结尾停在不协和和弦（如 vii°、II+）直接跳回 I
- 推荐：`... V | I` 或 `... IV | I`

**力度衔接：**
- 循环结尾的力度应接近循环开头的力度（避免 ff→pp 的跳变）
- 如果结尾有渐弱，开头也应从弱起
- 如果需要能量延续，结尾和开头都保持中强（mf）以上

**节奏衔接：**
- 循环结尾避免使用 fill 或突停（这些只在段落过渡时使用）
- 鼓组在循环最后 1 小节保持正常模式，不要加 fill

**实用检查：**
- 将结尾 2 小节 + 开头 2 小节连起来听，检查是否流畅
- 如果听到明显"断裂感"，调整结尾最后 1-2 拍的音高或和声

## 术语表

| 术语 | 说明 |
|------|------|
| 半音间隔 | 以根音为 0，向上半音数表示音程 |
| 罗马数字 | 大调和弦用大写（I II III…），小调和弦用小写（i ii iii…）。转位用斜线表示，如 C/E |
| 音程 | 两个音之间的距离，如大二度=2半音，小三度=3半音 |
| 调式 | 基于大调音阶不同起始音形成的音阶（如 Dorian、Lydian 等），详见「音阶定义」 |
| 特征音 | 与该调式最相关的音程，如 Lydian 的 #4 |
| MIDI 音符编号 | C4=60，每半音±1 |
| 动机 (Motif) | 旋律的最小构成单位（2-4 音符），具有辨识度和记忆点，详见「旋律写作技法」 |
| 终止式 (Cadence) | 和弦进行在乐句/段落结尾的收束方式（如 V→I 正格终止），详见「和弦进行库 → 终止式库」 |
| 转位 (Inversion) | 和弦的低音不为根音的排列方式。第一转位三音在低音，第二转位五音在低音 |
| 级进 | 相邻音符之间为二度（1-2 半音）的运动，旋律运动的基础 |
| 跳进 | 相邻音符之间为三度及以上（≥3 半音）的运动。5+ 半音为"大跳"，需特殊处理 |

## 输出格式约束（供 Agent 参考）

所有 Agent 输出 ABC 格式片段，由 Merger 合并为完整乐谱。格式规范见 theory.md 中的「ABC 记谱法参考」章节。

### 错误处理原则
- 若根音+音阶生成的音符超出所选乐器的音域，自动移调八度或选择最近可用音域。
- 若和弦进行中出现无法构建的和弦（如减三和弦音域过窄），可适当省略五音或重复根音。
- 若弯音超出音源范围，先发送 RPN 设置或限制弯音值在 ±8191 内。
