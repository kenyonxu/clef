# LLM 作曲使用指南

Clef 提供两种 LLM 辅助作曲方式。**Clef Compose** 是推荐方式，通过 Claude Code 的多 Agent 协作自动完成从需求到 MIDI 的全流程。**模板编曲**适用于没有 Claude Code 的用户，手动将 JSON 提交给任意 LLM。

---

## 方式一：Clef Compose（推荐）

在 Claude Code 中使用 `/clef-compose` 命令触发。你只需要用自然语言描述音乐需求，系统自动完成规划、创作、验证、迭代和输出。

### 工作流程

```
需求描述 → 音乐规划（plan.json） → 方向小样（用户确认） → 完整创作 → 质量评审 → 自动迭代 → 表现力注入 → MIDI 输出
```

全程有 3 个用户确认点：

1. **规划确认** — 系统展示调性、BPM、段落结构、配器方案，确认后继续
2. **方向小样确认** — 生成 4-8 小节的短片段供试听，确认方向正确后继续
3. **完整版确认** — 完整 MIDI 输出后，可试听并提出修改意见

### 描述需求时建议包含的要素

不需要懂乐理，把以下信息说清楚即可：

1. **场景/氛围** — 这段音乐用在游戏哪个部分？（战斗、探索、菜单、剧情、Boss 战等）
2. **情绪** — 紧张、轻松、激昂、神秘、温暖、悲伤、史诗……
3. **风格/参考** — 8-bit、管弦乐、电子、爵士、钢琴独奏，或"像《空洞骑士》的泪城"
4. **时长** — 30 秒、1 分钟、2 分钟，是否需要循环

如果想更精准，可以加上：
- **乐器偏好** — "主要用钢琴"、"不要鼓"、"像红白机那种声音"
- **节奏感** — "快节奏"、"慢悠悠"、"逐渐加速"

### 使用示例

```
/clef-compose 帮我写一段 boss 战斗音乐，D大调，140BPM，30秒，管弦风格

/clef-compose 森林探索背景音乐，长笛和弦乐，不需要鼓，90秒循环播放

/clef-compose 主菜单钢琴独奏，温暖安静，30秒，像星露谷物语的感觉
```

### 迭代修改

对结果不满意时，可以直接反馈，系统会根据反馈定位问题并修改：

- **明确反馈**（"B 段旋律太单调"）→ 系统直接修改对应声部
- **模糊反馈**（"某段听起来不对"）→ 系统生成分轨 Solo 文件，逐轨试听定位问题
- **全局反馈**（"整体不够紧张"）→ 系统调整整体规划参数

### 高级选项

| 参数 | 说明 | 示例 |
|------|------|------|
| `--ref` | 提供风格参考 MIDI 文件 | `--ref boss_theme.mid` |
| `--import` | 导入已有 ABC/MIDI 作为创作起点 | `--import draft.abc` |
| `--sf2` | 指定目标 SoundFont（自动适配音色特性） | `--sf2 GeneralUser-GS` |

## 方式二：模板编曲

如果你没有 Claude Code，可以将 [系统提示词](celf_composer_llm_system_prompt_cn.md) 和模板文件提交给 ChatGPT / Claude 等 LLM，手动完成 JSON → MIDI 转换。

### 使用步骤

1. 将 [templates/default.json](../../templates/default.json) 作为起始模板
2. 向 LLM 描述需求，附带系统提示词
3. LLM 返回 Clef JSON
4. 在 Godot 编辑器中右键 `.json` 文件 → **Convert to MIDI**
5. 在 Inspector 中预览试听

### 模板文件

| 文件 | 用途 |
|------|------|
| `templates/default.json` | 最小有效 JSON，适合快速测试 |
| `templates/example_full.json` | 完整示例，包含多轨、CC、弯音、速度变化 |
| `templates/llm_compose_guide.json` | 精炼规范速查表 |

## 描述示例速查

> **战斗音乐**：用在横版动作游戏的第一关，紧张刺激，速度快，大概 1 分钟

> **森林探索**：探索用的背景音乐，悠闲放松，慢一点，不需要鼓，长笛或者钢琴就行

> **Boss 战**：先安静，然后突然爆发，越来越快，2 分钟左右，要有震撼感

> **主菜单**：钢琴独奏，温暖安静，30 秒，循环播放

> **商店界面**：轻松愉快，中速，有一点爵士味，1 分钟
